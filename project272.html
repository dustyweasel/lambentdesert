This is a project I worked on in my freetime years ago. I was using Slackware Linux back then and I mostly used KDevelop, SDL, and some other stuff. These windows versions were hacked together later. It was mostly going to be a zelda-like platform game except I didn't want to use tiles. I wanted to be able to draw the world freehand and then simply scan it to give the whole game a pencil and notebook paper feel. You can even see the notebook paper on the first demo's beginning animation. Then I started experimenting and you can see a 3D space title menu on demo 2.5. Also note that I can't draw. If anyone with artistic talent did the artwork the demos would look a lot better.

I really didn't know anything when I started so I had to learn and hack away at it at the same time. Basically when I started I didn't even know what I didn't know. Now I know I need to remake the editor using the QT library or something.
